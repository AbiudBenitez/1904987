    Python


    Python es un lenguaje de programación de alto nivel que se utiliza para desarrollar aplicaciones de todo tipo. A diferencia de otros lenguajes como Java o .NET, se trata de un lenguaje interpretado, es decir, que no es necesario compilarlo para ejecutar las aplicaciones escritas en Python, sino que se ejecutan directamente por el ordenador utilizando un programa denominado interpretador, por lo que no es necesario “traducirlo” a lenguaje máquina.


    Simbolo

    Una de las operaciones más básicas cuando se trabaja con cadenas de caracteres es la concatenación. Esto consiste en unir dos cadenas en una sola, siendo el resultado un nuevo string.

    La forma más simple de concatenar dos cadenas en Python es utilizando el operador de concatenación +: 

              >>> hola = 'Hola'
          >>> python = 'Pythonista'
          >>> hola_python = hola + ' ' + python  # concatenamos 3 strings
          >>> print(hola_python)
          Hola Pythonista

    Operadores lógicos o booleanos

    A la hora de operar con valores booleanos, tenemos a nuestra disposición los operadores and, or y not. 

              >>> x = True
          >>> y = False
          >>> x or y
          True

    Operadores de comparación

    Los operadores de comparación se utilizan, como su nombre indica, para comparar dos o más valores. El resultado de estos operadores siempre es True o False. 

              >>> x = 9
          >>> y = 1
          >>> x < y
          False
          >>> x > y
          True
          >>> x == y
          False

    Operadores aritméticos en Python

    En cuanto a los operadores aritméticos, estos permiten realizar las diferentes operaciones aritméticas del álgebra: suma, resta, producto, división, … Estos operadores Python son de los más utilizados. 

              >>> x = 7
          >>> y = 2
          >>> x + y  # Suma
          9
          >>> x - y  # Resta
          5
          >>> x * y  # Producto
          14
          >>> x / y  # División
          3.5
          >>> x % y  # Resto
          1
          >>> x // y  # Cociente
          3
          >>> x ** y  # Potencia
          49

    Esqueletos de los progrmas

    Normalmente, el lenguaje de programacion de python no tiene un esqueleto en especifico, ya que, se puede programar sin la necesidad de tener algo antes de nuestro codigo, sin embargo si cuenta con normas de sintaxis las cuales se deben de respetar para poder programar, por ejemplo, se debe de tener unos espacios entre algunas de las lineas de codigo, debido a que estos representan cosas en el lenguaje de programacion a diferencia de otros que no necesitan estrictamente de estos.


    Aplicaciones a descargar para editar y ejecutar

    Para poder codificar en python bassta con tener su IDLE la cual se puede descargar desde la pagina oficial de python en la web, y desde esta se puede usar el bash para poder hacer codigo rapido o el IDLE directamente para poder crear archivos .py los cuales pueden ser procesados por la maquina.


    Tipo de paradigma

    Python es un lenguaje de programación multiparadigma. Esto significa que más que forzar a los programadores a adoptar un estilo particular de programación, permite varios estilos: programación orientada a objetos, programación imperativa y programación funcional. Otros paradigmas están soportados mediante el uso de extensiones.
