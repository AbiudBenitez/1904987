    PHP

    PHP (acrónimo recursivo de PHP: Hypertext Preprocessor) es un lenguaje de código abierto muy popular especialmente adecuado para el desarrollo web y que puede ser incrustado en HTML.

    En lugar de usar muchos comandos para mostrar HTML (como en C o en Perl), las páginas de PHP contienen HTML con código incrustado que hace "algo" (en este caso, mostrar "¡Hola, soy un script de PHP!). El código de PHP está encerrado entre las etiquetas especiales de comienzo y final que permiten entrar y salir del "modo PHP".


    Simbolo

    PHP admite diez tipos primitivos.

    Cuatro tipos escalares: boolean, integer, float y string

    Cuatro tipos compuestos: array, object, callable e interable

    Y finalmente dos tipos especiales: resource y NULL

    Puede que existan aún algunas referencias al tipo "double" en el manual. Considere al tipo double como float; los dos nombres existen sólo por razones históricas.

    El tipo de una variable usualmente no lo declara el programador; al contrario, es decidido en tiempo de ejecución por PHP dependiendo del contexto en el que se emplea dicha variable.


    Sintaxis

    Cuando PHP analiza un fichero, busca las etiquetas de apertura y cierre, que son , y que indican a PHP dónde empezar y finalizar la interpretación del código. Este mecanismo permite embeber a PHP en todo tipo de documentos, ya que todo lo que esté fuera de las etiquetas de apertura y cierre de PHP será ignorado por el analizador.

    PHP también permite la etiqueta de apertura abreviada

    Si un fichero contiene solamente código de PHP, es preferible omitir la etiqueta de cierre de PHP al final del mismo. Así se previene la adición de espacios en blanco o nuevas líneas accidentales después de la etiqueta de cierre, lo cual causaría efectos no deseados debido a que PHP comenzará la salida del búfer cuando no había intención por parte del programador de enviar ninguna salida en ese punto del script.


    Esqueletos de los programas

    Los programas en PHP son puro codigo incrustado en HTML asi que lo primero que se debe de poner en sus programas son las tipicas etiquetas html, despues de haberlas puesto, en el cuerpo del documento se puede iniciar codigo PHP con las etiquetas

    Quedaria algo asi:

              <html>
          <head>
          </head>
          <body>
            <?php
                //php code;
                //php code;
                //php code;
              ?>
          </body>
          </html>

    Aplicaciones a descargar para editar y ejecutar

    Para poder ejecutar programas en PHP se puede desde un software llamado apache el cual nos creara un servidor local en nuestra computadora por el cual podremos compilar paginas php y ver nuestros codigos ya ejecutados.

    Con este tambien se pueden hacer bases de datos con phpMyAdmin que es una manera sencilla de usar MySQL con PHP.


    Tipo de paradigma

    El lenguaje de programacion es multiparadigma y algunos de ellos son imperativo, funcional, orientado a objetos, procedural y reflexivo


    Ejemplo:

              <?php
            $un_bool = TRUE;   // un valor booleano
            $un_str  = "foo";  // una cadena de caracteres
            $un_str2 = 'foo';  // una cadena de caracteres
            $un_int  = 12;     // un número entero

            echo gettype($un_bool); // imprime: boolean
            echo gettype($un_str);  // imprime: string

            // Si este valor es un entero, incrementarlo en cuatro
            if (is_int($un_int)) {
                $un_int += 4;
            }

            // Si $un_bool es una cadena, imprimirla
            // (no imprime nada)
            if (is_string($un_bool)) {
                echo "Cadena: $un_bool";
            }
          ?>
